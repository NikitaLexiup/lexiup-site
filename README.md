# LexiUp site

Legal and support pages for the LexiUp iOS app, served by GitHub Pages at https://lexiup.space

- `privacy.html` - privacy policy (linked from the App Store listing, the in-app paywall and the AdMob GDPR message)
- `support.html` - support page (required by App Store review)
- `index.html` - landing

Edit, commit, push - GitHub Pages redeploys automatically.
